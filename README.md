### To install all dependencies run:
`npm install`

### To start simple server and frontend run:
`npm run dev`

### To test with coverage run:
`npm run coverage`

### To start react app run: 
`npm start`

### To start server run: 
`npm run serve`

### To test run:
`npm test`

### To build for production run:
`npm run build`

### To apply formatting: 
`npm run prittify`

### To apply check: 
`npm run lint`
